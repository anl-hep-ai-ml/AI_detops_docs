# MSL — Mars Science Laboratory

MSL is a multivariate time-series anomaly detection benchmark released by NASA's Jet Propulsion Laboratory, containing telemetry data from the Curiosity Mars rover. Like SMAP, it captures command and telemetry channels and includes anomalies verified by JPL engineers.

## Dataset Statistics

| Property | Value |
|---|---|
| Channels | 55 |
| Anomaly rate | 10.7% |
