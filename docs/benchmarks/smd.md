# SMD — Server Machine Dataset

SMD is a multivariate time-series dataset collected from 28 server machines at a large internet company (Alibaba). Each machine is monitored across 38 metrics (CPU, memory, network I/O, etc.) over multiple weeks. It is widely used as a benchmark for multivariate anomaly detection in operational settings.

## Dataset Statistics

| Property | Value |
|---|---|
| Machines | 28 |
| Channels per machine | 38 |
| Anomaly rate | ~5% |

## Citation

```
S. Su, Y. Zhao, Y. Niu, R. Liu, W. Sun, D. Pei.
```
