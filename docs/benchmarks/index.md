# Benchmarks

Methods are evaluated on publicly available benchmark datasets before being applied to experimental data. These datasets provide a standardised basis for comparing anomaly detection approaches.

| Dataset | Domain | Channels | Anomaly Rate |
|---|---|---|---|
| [SMAP](smap.md) | Spacecraft telemetry (NASA) | 25 | 13.1% |
| [MSL](msl.md) | Spacecraft telemetry (NASA) | 55 | 10.7% |
| [SMD](smd.md) | Server machine metrics | 38 | ~5% |
| [SWAT](swat.md) | Industrial control system | 51 | ~12% |
