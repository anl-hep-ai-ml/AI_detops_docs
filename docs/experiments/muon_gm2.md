# Muon g-2

The Muon g-2 experiment at Fermilab measures the anomalous magnetic moment of the muon to unprecedented precision, probing physics beyond the Standard Model via the discrepancy between the measured value and the Standard Model prediction.

## Application

Anomaly detection is applied to the electromagnetic calorimeter readout channels arranged in the 24-station storage ring. The time-series structure is fill-by-fill, aligned to the muon beam injection cycle. AD targets gain drift, beam scraping events, and kicker timing anomalies.
