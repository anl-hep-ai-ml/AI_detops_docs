# What is Agentic AI?

Agentic AI refers to systems where a language model acts as a reasoning engine that can plan, use tools, and take multi-step actions to accomplish a goal — rather than simply responding to a single prompt. In the context of detector operations, agentic systems open the door to automated diagnosis, adaptive monitoring, and knowledge-grounded decision support.

## Overview

The core building blocks of practical agentic systems include:

- **[Retrieval-Augmented Generation (RAG)](rag.md):** Grounding model outputs in domain-specific documents and data
- **[Model Context Protocol (MCP) Servers](mcp_servers.md):** Standardised tool interfaces that let agents interact with external systems
- **[Agentic Frameworks](agentic_frameworks.md):** Orchestration layers that coordinate multi-step reasoning and tool use
